# Space Suika
Space Suika is a 360 Degrees space adaptation of the popular suika game, in which the goal is to merge as many planets as possible. The game is playable on itch.io
## Building

Space Suika was entirely developped using Godot 4.5.1 stable, and can/should be modded using that exact version. It uses the Compatibility render.
